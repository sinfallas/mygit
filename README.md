A script to automate common tasks when using git

## How to use

```bash
curl -O https://raw.githubusercontent.com/sinfallas/mygit/master/mygit
chmod +x mygit
```

copy to /usr/bin/ and then execute:

```bash
mygit {up|change|change-all|refresh|up-all <message>}
```
